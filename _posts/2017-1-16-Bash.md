How to zip a folder?
`zip -r file_name.zip directory`

here `directory` is the location of all the files which we want to zip. and `-r` is the recursive option.

**How to copy from remote server to current server?**

scp is the command.
```
scp username@source-server-name:/which files to copy username@destination-server-name:/directory to copy
```
