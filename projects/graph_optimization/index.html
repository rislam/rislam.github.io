Query Optimization

Learning to optimize graph query

As a part of our course project for Structured Prediction and Advanced Database I worked on learning to optimize query plans for efficiently querying large-scale knowledge graphs. The ICAPS 2017 paper (Namaki et al., 2017) and ACML 2017 paper (Chowdhury et al., 2017) contains the primary results.

References

  1. Select-and-Evaluate :A Learning Framework for Large-Scale Knowledge Graph Search Chowdhury, F.A. Rezaur Rahman, Ma, Chao, Islam, Md Rakibul, Namaki, Mohammad Hossein, Faruk, Mohammad Omar, and Doppa, Janardhan Rao In Journal of Machine Learning Research (JMLR), Proceedings Track 2017
  2. Learning to Speed Up Query Planning in Graph Databases Namaki, Mohammad Hossein, Chowdhury, F. A. Rezaur Rahman, Islam, Md Rakibul, Doppa, Janardhan Rao, and Wu, Yinghui In Proceedings of 27th International Conference on Automated Planning and Scheduling (ICAPS) 2017
